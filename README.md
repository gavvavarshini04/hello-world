# Student Management System (Java)

## Description
A simple console-based Java application to manage student records.

## Features
- Add Student
- View Students
- Search Student by ID
- Delete Student

## Technologies Used
- Java
- ArrayList
- OOP Concepts

## How to Run
1. Compile the program:
   javac StudentManagementSystem.java

2. Run the program:
   java StudentManagementSystem
